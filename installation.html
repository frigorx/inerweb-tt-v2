Guide d'installation

1

Obtenir le fichier ZIP

Un fichier ZIP est comme une valise qui contient tous les fichiers du logiciel, compressés en un seul document. Vous le recevez soit par e-mail de votre administrateur, soit en le téléchargeant depuis la page GitHub du projet.

Une fois le fichier ZIP sur votre ordinateur (souvent dans le dossier Téléchargements), passez à l'étape suivante.

💡
Astuce : Si vous ne trouvez pas le fichier, ouvrez votre dossier Téléchargements en appuyant sur les touches Windows + E puis en cliquant sur "Téléchargements" dans le menu de gauche.
2

Extraire le fichier ZIP

Il faut maintenant "ouvrir la valise" pour sortir tous les fichiers. Faites un clic droit sur le fichier ZIP, puis choisissez "Extraire tout..." dans le menu qui apparaît. Cliquez ensuite sur "Extraire".

Vous devriez voir un dossier contenant plusieurs fichiers :

inerweb-tt-fe/
  ├─ index.html ← Page d'accueil
  ├─ inerweb_prof.html ← Espace professeur
  ├─ inerweb_admin.html ← Administration
  ├─ aide.html ← Guide complet
  └─ ... (autres fichiers)
💡
Astuce : Ne déplacez pas les fichiers individuellement hors du dossier. Ils doivent rester ensemble pour fonctionner correctement.
3

Ouvrir la page d'accueil

Dans le dossier extrait, double-cliquez sur le fichier index.html. Votre navigateur internet (Chrome, Firefox, Edge...) s'ouvrira automatiquement et affichera la page d'accueil du logiciel.

Si la page s'affiche correctement avec le logo et les boutons, tout fonctionne. Aucune installation supplémentaire n'est nécessaire.

💡
Astuce : Pour un accès rapide, faites un clic droit sur index.html puis "Envoyer vers > Bureau (créer un raccourci)". Vous aurez une icône directement sur votre bureau.
4

Configurer la connexion au serveur

Pour que vos données soient enregistrées, il faut connecter le logiciel au serveur. Ouvrez inerweb_prof.html, puis cliquez sur l'onglet Paramètres (icône engrenage).

Vous devez remplir deux champs (fournis par votre administrateur) :

1. L'URL du serveur API : c'est l'adresse du serveur qui stocke les données.
2. La clé API enseignant : c'est votre mot de passe de connexion au serveur.

Collez ces informations dans les champs correspondants, puis cliquez sur Enregistrer.

💡
Astuce : Si vous n'avez pas encore ces informations, contactez la personne qui vous a transmis le fichier ZIP. En attendant, vous pouvez utiliser le mode démo pour découvrir le logiciel.
5

Créer vos classes et ajouter les élèves

Dans l'espace professeur, allez dans l'onglet Classes. Vous pouvez créer une classe en lui donnant un nom (par exemple "2nde TNE A") puis ajouter les élèves de deux façons :

Manuellement : cliquez sur "Ajouter un élève" et tapez le nom et le prénom.
Par fichier CSV : préparez un fichier texte avec une ligne par élève au format NOM;PRENOM, puis importez-le.

💡
Astuce : Un fichier CSV peut être créé facilement avec Excel ou LibreOffice Calc. Mettez les noms en colonne A et les prénoms en colonne B, puis enregistrez au format CSV (séparateur point-virgule).
6

Générer et distribuer les QR codes

Chaque élève et chaque tuteur recevra un QR code unique pour accéder à son espace. Dans l'onglet QR Codes, sélectionnez une classe et cliquez sur "Générer". Vous obtiendrez une page imprimable avec tous les QR codes.

Imprimez la feuille et découpez les QR codes. Distribuez-en un à chaque élève. Ils n'auront qu'à le scanner avec leur téléphone pour accéder à leur espace personnel.

💡
Astuce : Plastifiez les QR codes ou collez-les dans les carnets de suivi. Un QR code abîmé ne fonctionnera plus. Vous pouvez toujours en régénérer un depuis l'interface.

Questions fréquentes

Vérifiez que vous cliquez bien sur index.html et non sur le fichier ZIP lui-même. Si le fichier s'ouvre dans un éditeur de texte (Bloc-notes), faites un clic droit dessus, choisissez "Ouvrir avec" puis sélectionnez votre navigateur (Chrome, Firefox ou Edge).
Ces informations sont fournies par l'administrateur qui a mis en place le serveur. Contactez la personne responsable dans votre établissement. En attendant, vous pouvez explorer le logiciel en mode démo (toutes les fonctionnalités sont accessibles, mais les données ne sont pas sauvegardées).
Téléchargez le nouveau fichier ZIP et extrayez-le. Vos données sont stockées sur le serveur, elles ne sont pas perdues. Il suffit de remplacer l'ancien dossier par le nouveau et de reconfigurer la connexion (étape 4). Vos classes et élèves seront toujours présents.
L'interface élève et tuteur est entièrement adaptée aux écrans de téléphones et tablettes. L'interface professeur fonctionne aussi sur tablette, mais un ordinateur est recommandé pour plus de confort lors de la saisie et la configuration.
Les données transitent de manière chiffrée entre votre navigateur et le serveur. Chaque utilisateur dispose d'une clé unique. Aucune donnée personnelle sensible n'est collectée : uniquement les noms, prénoms et les évaluations de compétences. L'administrateur peut consulter le journal d'accès à tout moment.
Cliquez sur le bouton "Mode démo" sur la page d'accueil, ou utilisez le lien ci-dessous. Le mode démo vous permet d'explorer toutes les fonctionnalités avec des données fictives, sans aucune configuration requise.
