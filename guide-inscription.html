Guide 1 — Inscription

Comment s'inscrire et recevoir son code d'accès

1
🔗
Ton prof t'envoie un lien
Tu reçois un message WhatsApp ou SMS avec un lien d'inscription.
Prof : Bonjour ! Inscris-toi ici :
https://frigorx.github.io/tt-ia/inscription.html
Clique sur le lien dans le message
⬇️
2
🔍
Cherche ton nom
La page s'ouvre. Tape les 3 premières lettres de ton nom de famille.
🔍 Tapez votre nom
MAR|
MARTIN Lucas — CAP IFCA 2
1 résultat
Clique sur ton nom quand il apparaît
Si ton nom n'apparaît pas, vérifie l'orthographe. Si ça ne marche toujours pas, préviens ton prof.
⬇️
3
📱
Entre ton numéro
Renseigne ton numéro de téléphone (obligatoire).
L'email est facultatif.
MARTIN Lucas — CAP IFCA 2
Téléphone *
06 12 34 56 78
Email (facultatif)
prenom.nom@email.fr
Valider mon inscription
Appuie sur "Valider mon inscription"
⬇️
4
⏳
Attends ton code
Tu vois un écran de confirmation. C'est fini pour toi !
Maintenant, attends que ton prof t'envoie ton code personnel.
✅
Inscription enregistrée !
Votre enseignant va vous envoyer
votre code d'accès personnel.
Combien de temps ? Ton prof vérifie les inscriptions et t'envoie le code par WhatsApp ou SMS. Ça peut prendre de quelques minutes à 24h.
⬇️
5
🔑
Tu reçois ton code !
Ton prof t'envoie un message avec ton code et un lien direct.
Prof : Bonjour Lucas !

Ton code inerWeb : ABCD1234

Clique ici pour te connecter :
https://frigorx.github.io/tt-ia/inerweb_eleve.html?token=ABCD1234
Clique sur le lien pour ouvrir ton espace directement.
Ou bien tape le code sur la page de connexion.
🎉

Bravo, tu es inscrit !
