👨‍🏫 Espace Professeur

▼
📋 À quoi ça sert ?

L'espace professeur est le cœur de la plateforme. Il permet de :

  • Gérer les élèves — fiches complètes, classes, entreprises de stage
  • Évaluer les compétences — E31 (Réalisation), E32 (Diagnostic), E33 (Dossier)
  • Suivre les PFMP — périodes de stage, tuteurs, progression
  • Générer les bilans — notes certificatives, jurys, exports PDF
  • Partager des documents — conventions, supports, rapports-types

📥 Exports et bilans

L'espace professeur permet de générer plusieurs types d'exports :

  • Fiche élève PDF — bilan individuel avec toutes les compétences et niveaux
  • Grille de jury — document formaté pour les commissions d'évaluation
  • Export Excel — tableau de données brutes pour analyse
  • Backup JSON — sauvegarde complète de toutes les données
  • QR codes élèves — badges individuels pour scanner
✅
Les exports fonctionnent hors-ligne — ils utilisent les données stockées localement dans votre navigateur.

🏢 Espace Tuteur entreprise

▼
📋 À quoi ça sert ?

L'espace tuteur permet au tuteur de stage en entreprise d'évaluer le stagiaire directement depuis son téléphone, sans créer de compte. Il sert à :

  • Évaluer les compétences techniques du stagiaire (12 compétences E31)
  • Noter le comportement professionnel (ponctualité, initiative, sécurité...)
  • Transmettre l'évaluation à l'enseignant automatiquement
⏱️
L'évaluation prend environ 5 à 10 minutes. Aucun compte à créer, aucune installation.
🔑 Comment accéder

Le tuteur accède à l'application de 2 façons :

Scanner le QR code du badge tuteur (carte verte) remis par l'enseignant — la caméra du téléphone s'ouvre automatiquement
Ou saisir manuellement le code stagiaire et le token tuteur sur la page de scan

📓 Le journal de stage

Le journal est l'outil le plus important de ton stage. Chaque jour, ajoute tes activités :

Va dans l'onglet Journal (ou clique sur la bannière d'accueil)
Prends des photos de ton travail (max 10 par jour) — pas de visages ni d'infos confidentielles
Choisis le type d'entrée : activité, observation, question ou réflexion
Décris ce que tu as fait ou appris
Clique Enregistrer — c'est synchronisé automatiquement
📷
Règles photos : ✅ Ton travail, le matériel, les installations · ❌ Pas de visages · ❌ Pas d'infos clients · 📏 Max 10 photos/jour

🔐 Administration

▼
📋 À quoi ça sert ?

L'espace administration permet de gérer la plateforme au niveau système :

  • Gérer les utilisateurs — créer, modifier, désactiver des comptes (admin, enseignant, lecture)

  • Gérer les clés système — rotation des clés API et admin
  • Consulter le journal — traçabilité des actions d'administration

🎫 Tokens élèves et tuteurs

Les tokens sont les identifiants uniques qui permettent aux élèves et tuteurs d'accéder à leurs espaces respectifs.

Allez dans l'onglet 🎫 Tokens
Sélectionnez l'élève dans la liste déroulante
Cliquez 🔄 Générer nouveaux tokens
Copiez les tokens générés — le token élève (badge bleu) et le token tuteur (badge vert)
⚠️
La génération de nouveaux tokens invalide les anciens. L'élève et le tuteur devront utiliser les nouveaux codes.

❓ Questions fréquentes

▼
Je n'ai pas ma clé API / mon code, que faire ? ▼
Professeur : Contactez l'administrateur de votre établissement pour obtenir la clé API.
Élève : Demandez votre badge QR à votre professeur principal.
Tuteur : Le badge vert vous est remis par l'enseignant au début de la PFMP.
L'application fonctionne-t-elle sans internet ? ▼
Oui, partiellement. Les données sont sauvegardées localement dans votre navigateur. Vous pouvez évaluer, ajouter des entrées au journal, etc. Les données seront synchronisées automatiquement dès que la connexion revient. Seul le premier chargement nécessite une connexion.
J'ai perdu mes données, que faire ? ▼
Les données sont stockées dans votre navigateur ET sur le serveur Google Sheets. Si vous avez vidé votre navigateur, reconnectez-vous — vos données serveur seront rechargées. Pour le professeur, utilisez la fonction Restaurer backup si vous avez exporté un fichier JSON.
Le QR code ne se scanne pas, que faire ? ▼
Vérifiez que votre navigateur a l'autorisation d'accéder à la caméra. Si le scan échoue, utilisez la saisie manuelle en entrant le code et le token imprimés sous le QR code.
Quel navigateur utiliser ? ▼
Chrome ou Edge sont recommandés (ordinateur et mobile). Safari fonctionne aussi sur iPhone/iPad. Évitez les navigateurs très anciens — l'application utilise des fonctionnalités modernes.
Comment exporter mon journal en PDF ? (Élève) ▼
Sur la page d'accueil, clique sur 📄 Export PDF. Une nouvelle fenêtre s'ouvre avec ton journal formaté — utilise le bouton 🖨️ Imprimer et choisis "Enregistrer en PDF" comme destination.
Le tuteur peut-il modifier son évaluation après l'envoi ? ▼
Oui, tant que l'enseignant n'a pas validé l'évaluation. Le tuteur peut rescanner le badge et modifier ses réponses. Les évaluations sont envoyées avec le statut "proposé" — seul le professeur peut les valider définitivement.
Comment contacter le support ? ▼
Contactez l'enseignant référent de votre établissement. Pour les problèmes techniques, signalez-les sur le dépôt GitHub du projet.
